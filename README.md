# weather-app

A modern and responsive Weather Application that provides real-time weather information for any location. The app displays temperature, weather conditions, humidity, and wind speed with a clean UI and dynamic gradient backgrounds that change according to the weather.

This project is beginner-friendly and built to practice API integration, responsive design, and modern UI styling.
